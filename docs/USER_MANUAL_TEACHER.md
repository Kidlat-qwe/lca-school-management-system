## Introduction

This manual is specifically designed for **Teacher** users of the Physical School Management System. As a Teacher, you have access to manage your assigned classes, mark attendance, view student information, and access curriculum materials.

### Key Characteristics of Teacher Role

- **Access Level**: Assigned classes only
- **Primary Functions**: 
  - Manage class attendance
  - View student information
  - Access curriculum and program details
  - View class schedules
  - Read announcements
- **Limitations**: 
  - Cannot create or modify classes
  - Cannot manage enrollments
  - Cannot access financial information
  - Cannot modify curriculum (view only)
  - Only see classes you're assigned to

## Getting Started

### Accessing the System

1. Navigate to your school's system URL
2. Enter your email and password
3. Click "Login"
4. You'll be redirected to the Teacher Dashboard

### First Time Setup

1. Ensure your account is created by Admin/Superadmin
2. Verify you're assigned to the correct classes
3. Review your Dashboard for assigned classes
4. Familiarize yourself with the Calendar and Classes pages

## Teacher Role Overview

### What You Can Do

- View your assigned classes
- Mark attendance for class sessions
- View student lists and information
- Access curriculum and program details
- View class schedules in Calendar
- Read school announcements
- View session details and curriculum topics

### What You Cannot Do

- Create or edit classes
- Enroll or unenroll students

### Classes

**Path**: Classes

#### Purpose

View and manage your assigned classes. This is your main page for class management, attendance, and student information.

#### Features

**Viewing Your Classes**

- See all classes you're assigned to
- Class information:
  - Class name and code
  - Program name
  - Room location
  - Schedule (days and times)
  - Number of students
  - Status

**Class List View**

- Table showing all your classes
- Columns:
  - Class Code
  - Class Name
  - Program
  - Room
  - Student Count
  - Schedule
  - Actions

**Viewing Class Details**

1. Click on a class name or "View" button
2. See comprehensive class information:

**Class Overview Section**
- Class name and code
- Program information
- Room location
- Schedule (days and times)
- Teacher(s) assigned
- Start and end dates
- Maximum students and current enrollment

**Sessions Section**
- All class sessions organized by phase
- Expandable/collapsible phases
- Each session shows:
  - Session number
  - Date and time
  - Room
  - Status (Scheduled, Completed, Cancelled, Rescheduled, In Progress)
  - Attendance status
  - Curriculum topic
- Click phase number to expand/collapse sessions

**Students Section**
- List of enrolled students
- Student information:
  - Name
  - Email
  - Phase enrolled
  - Enrollment date
- Students with "Pending (Downpayment Paid)" status:
  - Have paid downpayment
  - Not yet enrolled in Phase 1
  - Will be enrolled when first installment is paid

**Attendance Section**
- View attendance records
- See attendance statistics
- Track attendance history

**Curriculum Section**
- View curriculum for the class
- See phase and session topics
- Access learning goals and agendas

**Marking Attendance**

This is a primary function for teachers.

1. **Navigate to Class**: Go to Classes page, click on your class
2. **Find Session**: Locate the session you want to mark attendance for
   - Can mark attendance for today's or past sessions
   - Future sessions cannot have attendance marked
3. **Click "Mark Attendance"**: 
   - Button on session row
   - Or three dots menu → "Mark Attendance"
4. **Attendance Modal Opens**:
   - Shows all students enrolled in that phase
   - Student list with checkboxes/buttons
5. **Mark Each Student**:
   - **Present**: Student attended class
   - **Absent**: Student did not attend
   - **Late**: Student arrived late
   - **Excused**: Absence is excused (with valid reason)
6. **Add Session Information** (optional):
   - **Notes**: Add notes about the session
   - **Agenda**: Document what was covered
7. **Quick Actions**:
   - "Mark All Present" button: Quickly mark all students as present
8. **Save Attendance**:
   - Click "Save Attendance" button
   - Attendance is recorded
   - **Important**: Once saved, attendance cannot be edited (prevents tampering)

**Attendance Features**

- **Attendance Lock**: Once saved, attendance is locked and cannot be edited
- **Session Notes**: Add notes about the session (student behavior, topics covered, etc.)
- **Session Agenda**: Document what was planned/covered
- **Attendance History**: View past attendance records
- **Attendance Status Indicators**:
  - Present: Checkmark (green)
  - Absent: X mark (red)
  - Late: Clock icon (yellow)
  - Excused: Info icon (blue)

**Viewing Session Details**

1. Click on a session
2. See session information:
   - Date and time
   - Room location
   - Curriculum topic
   - Learning goals
   - Agenda
   - Students enrolled
   - Attendance status
   - Session notes

**Session Management**

- **View Session**: Click on session to see details
- **Mark Attendance**: Primary function for sessions
- **View Curriculum**: See curriculum topic for session
- Sessions are automatically created by the system
- Cannot create or delete sessions

**Student Information**

- View student list for each class
- See student details:
  - Full name
  - Email
  - Phone number (if available)
  - Phase enrolled
  - Enrollment status
- Students are enrolled by Admin/Superadmin
- Teachers cannot enroll or unenroll students

## Daily Teaching Workflow

### Before Class

**Preparation Steps:**

1. **Check Dashboard**
   - Review today's schedule
   - See which classes you have
   - Check room assignments

2. **Review Calendar**
   - Confirm session times
   - Verify room locations
   - Check for schedule changes

3. **Check Announcements**
   - Read any new announcements
   - Note important updates
   - Follow any instructions

4. **Review Curriculum**
   - Check curriculum for today's sessions
   - Review learning goals
   - Prepare materials based on agenda

5. **Prepare Materials**
   - Gather teaching materials
   - Prepare handouts (if needed)
   - Set up classroom

### During Class

**Class Activities:**

1. **Take Attendance**
   - Mark students present/absent
   - Note late arrivals
   - Record excused absences
   - Save attendance immediately

2. **Follow Curriculum**
   - Cover planned topics
   - Address learning goals
   - Follow agenda structure

3. **Document Session**
   - Add session notes if needed
   - Note any issues or achievements
   - Document deviations from plan

### After Class

**Post-Class Tasks:**

1. **Save Attendance**
   - Ensure attendance is saved
   - Verify all students marked
   - Check for any errors

2. **Add Session Notes** (if needed)
   - Document what was covered
   - Note student participation
   - Record any issues

3. **Update Session Status**
   - Session is automatically marked as completed when attendance is saved
   - Verify status is updated

4. **Prepare for Next Class**
   - Review next session's curriculum
   - Prepare materials
   - Check schedule

## Attendance Management

### Marking Attendance - Step by Step

**Step 1: Navigate to Class**
1. Go to Classes page
2. Find the class you want
3. Click on class name or "View" button

**Step 2: Find Session**
1. Scroll to Sessions section
2. Expand the phase containing the session
3. Find today's session (or past session)
4. Sessions are organized by date

**Step 3: Open Attendance Modal**
1. Click "Mark Attendance" button on session row
2. Or click three dots menu → "Mark Attendance"
3. Attendance modal opens
4. Student list appears

**Step 4: Mark Each Student**
1. For each student, select status:
   - **Present**: Click "Present" or checkmark
   - **Absent**: Click "Absent" or X mark
   - **Late**: Click "Late" or clock icon
   - **Excused**: Click "Excused" and may need to add reason
2. **Quick Action**: Click "Mark All Present" to mark everyone present at once, then adjust as needed

**Step 5: Add Session Information (Optional)**
1. **Session Notes**: Add any notes about the session
   - Student behavior
   - Topics covered
   - Issues or achievements
2. **Session Agenda**: Document what was planned/covered
   - Lesson topics
   - Activities
   - Materials used

**Step 6: Save Attendance**
1. Review all attendance marks
2. Click "Save Attendance" button
3. Attendance is recorded and locked
4. **Important**: Cannot edit after saving

### Attendance Status Options

- **Present**: Student attended the full session
- **Absent**: Student did not attend
- **Late**: Student arrived after session started
- **Excused**: Student's absence is excused (with valid reason)

### Attendance Best Practices

1. **Mark Promptly**: Mark attendance at the start or end of class
2. **Be Accurate**: Ensure correct status for each student
3. **Use Notes**: Add notes for important observations
4. **Save Immediately**: Save attendance as soon as marked
5. **Review Before Saving**: Double-check all marks before saving
6. **Document Excused Absences**: Note reasons for excused absences

### Viewing Attendance History

- Click on class
- View Sessions section
- See attendance status for each session
- Attendance indicators show:
  - Present: Green checkmark
  - Absent: Red X
  - Late: Yellow clock
  - Excused: Blue info icon

### Attendance Reports

- View attendance statistics in class details
- See attendance rates per student
- Track patterns (frequent absences)
- Monitor overall class attendance

## Troubleshooting

### Common Issues

**Problem**: Cannot see my classes
- **Solution**: 
  - Verify you're assigned to classes (contact Admin)
  - Refresh the page
  - Check you're logged in as Teacher
  - Contact Admin if classes are missing

**Problem**: Cannot mark attendance
- **Solution**:
  - Verify session is today or past date (cannot mark future sessions)
  - Check you're assigned to the class
  - Refresh the page
  - Contact Admin if issue persists

**Problem**: Attendance already marked (cannot edit)
- **Solution**: This is expected - attendance is locked after saving
  - Contact Admin if correction is needed
  - Attendance locking prevents tampering

**Problem**: Cannot see student information
- **Solution**:
  - Check Student List page
  - Verify students are enrolled in your classes
  - Contact Admin if student information is missing

**Problem**: Curriculum not showing
- **Solution**:
  - Verify program has curriculum defined
  - Check Curriculum page
  - Contact Admin if curriculum is missing

**Problem**: Schedule not showing correctly
- **Solution**:
  - Check Calendar page
  - Verify class dates are correct
  - Refresh the page
  - Contact Admin if schedule is wrong

**Problem**: Cannot access certain pages
- **Solution**:
  - Some pages are not available to Teachers (this is expected)
  - Teachers have limited access by design
  - Contact Admin for access to additional features

### Getting Help

**For Class Issues**:
- Contact your branch Admin
- Report class scheduling problems
- Request student information access

**For Technical Issues**:
- Contact IT support
- Report system errors
- Request feature assistance

**For Teaching Support**:
- Contact curriculum coordinator
- Reach out to other teachers
