## Demo order (recommended)

1. **Part 1** — Login, dashboard, branch setup (students, classes, packages)
2. **Part 2** — Enrollment, invoices, record payment
3. **Part 3** — Payment Logs, corrections, acknowledgement receipts
4. **Part 4** — End of Shift (EOD) and Cash Deposit
5. **Part 5** — Daily Summary Sales tracking (what happens after submit)

Use a **Branch Admin** test account assigned to **one branch** only. For verification steps (EOD approve, cash deposit approve), prepare a second login: **Superfinance** or **Finance** (EOD only).
